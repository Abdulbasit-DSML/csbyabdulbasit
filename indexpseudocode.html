📖 Quick Reference — IGCSE Pseudocode Syntax

📌 Variables & Constants

DECLARE Counter : INTEGER
DECLARE Name : STRING
DECLARE Price : REAL
DECLARE Grade : CHAR
DECLARE Done : BOOLEAN
CONSTANT Pi ← 3.14159

📌 Assignment & I/O

Counter ← 0
Counter ← Counter + 1
INPUT Name
OUTPUT "Hello, ", Name
OUTPUT "Value: ", Counter

📌 Arithmetic Operators

+   Addition
-   Subtraction
*   Multiplication
/   Division
^   Power (e.g. X ^ 2)
MOD(10, 3)   → 1
DIV(10, 3)   → 3

📌 Relational & Boolean

=   Equal to
<>  Not equal to
<   Less than
<=  Less than or equal
>   Greater than
>=  Greater or equal
AND  OR  NOT

📌 IF Statements

IF Score > 50
  THEN
    OUTPUT "Pass"
  ELSE
    OUTPUT "Fail"
ENDIF

IF X > 90
  THEN
    OUTPUT "A"
  ELSEIF X > 70
    THEN
      OUTPUT "B"
  ELSE
    OUTPUT "C"
ENDIF

📌 CASE Statement

CASE OF Grade
  'A' : OUTPUT "Excellent"
  'B' : OUTPUT "Good"
  'C' : OUTPUT "Average"
  OTHERWISE OUTPUT "Unknown"
ENDCASE

📌 Loops

// FOR loop
FOR i ← 1 TO 10
    OUTPUT i
NEXT i

// WHILE loop
WHILE Count < 10 DO
    Count ← Count + 1
ENDWHILE

// REPEAT UNTIL
REPEAT
    INPUT Password
UNTIL Password = "abc"

📌 Arrays

// 1D Array
DECLARE Nums : ARRAY[1:5] OF INTEGER
Nums[1] ← 42
OUTPUT Nums[1]

// 2D Array
DECLARE Grid : ARRAY[1:3,1:3] OF INTEGER
Grid[1,1] ← 99
OUTPUT Grid[1,1]

📌 Functions & Procedures

FUNCTION Add(A:INTEGER, B:INTEGER) RETURNS INTEGER
    RETURN A + B
ENDFUNCTION

OUTPUT Add(3, 4)

PROCEDURE Greet(Name:STRING)
    OUTPUT "Hello, ", Name
ENDPROCEDURE

CALL Greet("Ali")

📌 String Built-ins

LENGTH("Hello")          → 5
UCASE("hello")          → "HELLO"

SUBSTRING("Hello",1,3)  → "Hel"

📌 Math Built-ins

ROUND(3.756, 2)  → 3.76

MOD(10, 3)       → 1
